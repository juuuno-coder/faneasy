이름 수정 기능이 추가되었습니다!

**사용 방법:**

1. 최고관리자로 로그인
2. 회원 관리 탭으로 이동
3. 회원 이름을 **더블클릭**하면 입력창으로 변경됩니다
4. 새 이름을 입력하고 **Enter**를 누르면 저장됩니다
5. **ESC**를 누르면 취소됩니다

**다음 단계:**
이제 Firebase Authentication의 사용자들을 자동으로 Firestore와 동기화하는 기능을 추가하겠습니다.

이를 위해 다음 작업이 필요합니다:

1. Firebase Functions 또는 Cloud Run을 사용한 자동 동기화
2. 또는 관리자 페이지에서 수동으로 "동기화" 버튼을 눌러 Auth 사용자들을 Firestore에 추가

어떤 방식을 선호하시나요?

- A: 자동 동기화 (새 사용자 가입 시 자동으로 Firestore에 추가)
- B: 수동 동기화 버튼 (관리자가 버튼을 눌러 Auth 사용자들을 Firestore에 일괄 추가)
